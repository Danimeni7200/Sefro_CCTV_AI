================================================================================
                         WHAT THIS DOES
================================================================================

1. Removes the old build_full directory (which has corrupted CMake cache)
2. Creates a fresh build_full directory
3. Runs CMake to configure the build
4. Compiles the C++ code
5. Creates a new executable

This fixes the "target already exists" error.
